This project features an automated DevSecOps CI/CD Security Pipeline designed to monitor and enforce code security and quality standards in real-time using GitHub Actions.

Key Features
Automated Scanning: The pipeline triggers an automatic security scan on every code push.

Security & Vulnerability Assessment: Utilizes Bandit for security analysis and Safety to check for vulnerabilities in project dependencies.

Code Quality Enforcement: Implements Flake8 to ensure Python code adheres to professional quality standards.

Tech Stack
Languages: Python

CI/CD: GitHub Actions

Security Tools: Bandit, Safety, Flake8
